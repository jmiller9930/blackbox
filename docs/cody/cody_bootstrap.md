# Cody bootstrap (Phase 1)

**Phase 1 only creates Cody** — prompts, identity, and a minimal Python runtime skeleton with planning/reporting hooks.

Later phases are expected to **use Cody** as the engineering agent to help design and implement additional agents (**Billy**, **Robbie**, **Bobby**) and broader platform capabilities. Those agents are **not** implemented in Phase 1; names are placeholders until their phases define roles and interfaces.
