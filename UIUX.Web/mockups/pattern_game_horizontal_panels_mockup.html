5. Layout Notes

Why this three-panel top row reads more cleanly than the current mixed layout.

Notes
Panel Roles
Panel 1 = controls
Panel 2 = scorecard
Panel 3 = modules online
Intent
The top header answers system-level questions at a glance. The first row is your operator command row. The second row is the evidence row. This keeps “configure”, “monitor”, and “inspect” from colliding visually.
This mockup is intentionally blocky. It is meant to help us agree on spatial hierarchy first before we talk about detailed styling, tabs, or exact control placement.
